# CSRBF_FSKS
The main program CSRBFs_FSKS.m adopts the compactly supported radial basis function (CSRBF) method to solve the three-dimensional three-temperature radiation diffusion equation. Relevant subroutines are presented in Reference [1], and the Halton nodes used in this work can be constructed conveniently.

[1]Fasshauer G E. Meshfree Approximation Methods with MATLAB. Singapore: World Scientific Publishers, 2007
